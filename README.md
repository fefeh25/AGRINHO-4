# 🌱 Agro Forte - Futuro Sustentável

Uma landing page moderna e responsiva desenvolvida com foco no tema **Agro Forte: Futuro Sustentável**, promovendo o equilíbrio entre produtividade agrícola, inovação tecnológica e preservação ambiental.

---

## 📖 Sobre o Projeto

O projeto foi criado para representar visualmente a evolução do agronegócio sustentável, destacando:

* Produção agrícola inteligente
* Uso eficiente da água
* Preservação ambiental
* Energia limpa
* Agricultura de precisão
* Desenvolvimento sustentável

A proposta é transmitir uma mensagem de inovação, responsabilidade ambiental e crescimento econômico sustentável.
